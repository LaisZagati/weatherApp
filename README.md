# 🌤 Weather App

An interactive and beginner-friendly weather interface built with **HTML**, **CSS**, and **JavaScript**. This app simulates live weather updates and encourages user interaction through simple prompts, displaying the current temperature and a short-term forecast.

---

### 🧩 Project Highlights

- ✅ Displays current weather for a default city (Tokyo)
- ✅ Includes a 3-day weather forecast
- ✅ Users can **update the city and temperature** dynamically
- ✅ Responsive design with hover effects and smooth transitions
- ✅ Uses JavaScript DOM manipulation and event handling

### ⚙️ How It Works

When users click the **"Change city"** button:

1. A prompt asks for the user's city  
2. A second prompt asks for the current temperature  
3. The app updates the heading with:  
   - A relevant emoji (🌤, 😀, 🥶)  
   - The current temperature  
   - The city name
